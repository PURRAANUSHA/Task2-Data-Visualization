# 🚗 Task 2: Data Visualization Dashboard

## 📊 Project Title

Car Prices Dashboard

## 📁 Dataset

This project uses the **Car Prices dataset** obtained from Kaggle. The dataset contains information about car brands, selling prices, transmission types, and other attributes.

## 🎯 Objective

The objective of this project is to analyze car price data and create an interactive dashboard using Power BI to gain meaningful insights.

## 📌 Dashboard Features

* Selling Price by Car Brand
* Selling Price by State
* Cars Count by Transmission
* Average Selling Price by Transmission
* Selling Price Trends by Year

## 🛠 Tools Used

* Power BI
* Data Cleaning
* Data Visualization

## 📷 Dashboard Preview

![Dashboard](outputs/dashboard.png)

## 📈 Insights

* Selling prices vary significantly across different car brands, indicating differences in brand value and demand.
* Certain states show higher average selling prices, suggesting regional market variations.
* Automatic transmission cars appear more frequently in the dataset.
* Car prices fluctuate across different years, showing market trends and depreciation.
* Some brands maintain higher resale value compared to others.

## 📌 Conclusion

This dashboard helps in understanding car pricing patterns and market trends. It provides an interactive way to explore the data and supports better decision-making.
